A) Import project in Android studio

Step 1:- Select ‘Open an existing Android Studio project’ option

Step 2:- Select QIBus project then click ok button and wait for few minute

B) How to change applicationId

Step 1:- How to Change applicationId Find build.gradle file inside app folder and click to open and change your applicationId

Note : No need to change package name src-> main-> package name, change only applicationId for play store upload

C) How to change application menu icon, background and app icon

Open res folder inside open all drawable folder and all mipmap folder and replace your background images and icons here

D) How to change application name

Open values folder inside Strings.xml file and change your "app_name" here

E) How to change app font

Step 1:- Click and open font folder and change ttf file

F) How to change color

Step 1:- Click and open color.xml file
